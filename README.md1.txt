# 📊 Superstore Sales Dashboard - Excel Project

## 📁 Project Overview
This project aims to visualize and analyze sales performance for a fictional superstore using Microsoft Excel. It helps identify top-performing segments, regions, and trends using a dynamic and interactive dashboard.

---

## 🧾 Dataset
- **Source:** Superstore Sales Dataset (Kaggle / Tableau Sample)
- **Records:** +9,000 transactions
- **Fields:** Order ID, Date, Region, Segment, Sales, Profit, Category, Sub-Category

---

## 🎯 Goals
- Create a user-friendly dynamic dashboard.
- Monitor key KPIs (Sales, Profit, Orders).
- Highlight trends by Region, Category, and Time.

---

## 🛠 Tools & Features
- Excel Pivot Tables
- Slicers & Timelines
- Conditional Formatting
- Charts (Bar, Line, Pie)
- Year-over-Year (YoY) KPI Calculations

## 💡 Key Insights
- 📈 The West region generates the highest sales.
- 💰 Technology is the most profitable category.
- 🧾 Orders spike during November and December.

## 🔍 Bonus Features
- YoY dynamic metrics with custom formulas.
- Filter by Segment, Region, and Time Period.
